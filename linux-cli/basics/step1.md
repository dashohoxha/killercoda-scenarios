# First commands

1. Display the current date and time:

   `date`{{execute}}
   
   `date +%Y-%m-%d`{{execute}}
   
2. Display a calendar of the current month:

   `cal`{{execute}}
   
   For another month:
   
   `cal 5 2020`{{execute}}
   
3. Check how much free space there is on the disk drives:

   `df`{{execute}}
   
   `df -h`{{execute}}
   
   `df -h /`{{execute}}
   
4. Display the amount of the free memory:

   `free`{{execute}}
   
   `free -h`{{execute}}
